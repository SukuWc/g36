# Padlófűtés rétegrend
_Érintett helyiségek: előszoba, gyerekszoba, fürdőszoba_

## Rétegrend (felülről lefelé)

| Réteg | Vastagság | Állapot |
|---|---|---|
| Burkolat | 2cm | — |
| Esztrich beton | 5cm | — |
| Padlófűtés rendszerlemez | 7cm | — |
| XPS hőszigetelés | 15cm | — (csatornacső ebben fut) |
| Bitumenes vízszigetelés | 1cm | anyag beszerezve, felrakás hátra |
| Szerelőbeton | 8cm | ✅ kész |
| **Összesen** | **38cm** | |

## Megjegyzések
- A csatornacső a vízszigetelés felett, az XPS hőszigetelésbe ágyazva fut
- A csatornacső mélységi mozgástere: 15cm (XPS vastagsága)
- A padló végső kész szintje: +30cm a meglévő szerelőbeton felett
